# Forge — Setup Video Script
Target length: ~60-75 seconds. Read at a normal pace, pause briefly at each ✂️ cut point if editing.

---

**[0:00 – Show the Forge site homepage]**
"This is Forge — it rewrites rough prompts into sharp ones, right inside Claude, ChatGPT, and Gemini. Let me show you how to set it up. It takes about a minute."

✂️

**[0:08 – Click "Download for Chrome"]**
"Click Download — this saves the extension straight from the site, no GitHub, no extra steps."

✂️

**[0:14 – Show the downloaded zip in Downloads/Finder, unzip it]**
"Unzip the file you just downloaded. You'll get a folder — keep track of where it is, you'll need it in a second."

✂️

**[0:22 – Open a new tab, type chrome://extensions, hit enter]**
"Open a new tab and go to chrome colon slash slash extensions."

✂️

**[0:28 – Toggle "Developer mode" top right]**
"Turn on Developer mode, top right corner."

✂️

**[0:33 – Click "Load unpacked"]**
"Click Load unpacked."

✂️

**[0:37 – File picker opens, navigate to and select the unzipped folder]**
"Find the folder you unzipped — the one with manifest.json inside it — and select it."

✂️

**[0:45 – Forge icon appears in the extensions list / toolbar]**
"And there it is — Forge is installed."

✂️

**[0:49 – Click the Forge toolbar icon, popup opens]**
"Click the icon in your toolbar to open the settings."

✂️

**[0:53 – Show console.groq.com/keys in another tab briefly, copy a key]**
"You'll need a free Groq API key — console dot groq dot com slash keys, sign up, generate a key, copy it. No credit card needed."

✂️

**[1:02 – Paste key into the Forge popup, click Save]**
"Paste it into Forge, hit Save."

✂️

**[1:06 – Switch to claude.ai, type a rough prompt]**
"Now head to Claude, type something rough — like 'help me write an email about a delay.'"

✂️

**[1:12 – Click the floating ⚡ button, preview modal appears]**
"Click the spark button, bottom right. It rewrites your prompt in a couple seconds."

✂️

**[1:18 – Show the preview, click "Use this"]**
"Review it, edit if you want, click Use this — and it drops straight into the input box."

✂️

**[1:23 – End card / send the message]**
"That's it. Same thing works on ChatGPT and Gemini too."

## Recording tips
- Loom (free, browser-based) or QuickTime (Mac, Cmd+Shift+5) both work fine for this length
- Record at 1280×720 or higher, crop to just the browser window if possible — cleaner than full desktop
- Do the Groq key step with a **test key you're about to delete**, or blur/cut that section — don't leave a real key visible in a public video
- One clean take beats a perfectly narrated one — small stumbles are fine, re-record only if you lose your place badly
